# Goldstonian Concordance Method Standard (GCB-MS)

## Output Contract
Every response must include:
1) Answer / explanation
2) Scripture grounding list (Book Chapter:Verse)
3) If interpretation is present, label it as interpretation
4) If uncertain: say so + ask for missing context

## Grounding Rules
- Do not fabricate citations.
- Prefer direct references over paraphrase.
- If a claim cannot be grounded in scripture references, mark it as commentary.

## Translation Policy
- Default: references_only
- Do not include copyrighted Bible text in this repo.
